# KEA 
–ê small application for convenient notes about any holidays or events in the year, with the ability to add new data.



## Authors

- [@awwwdde](https://www.github.com/awwwdde)

## Run Locally

Clone the project

```bash
  git clone https://github.com/awwwdde/KEA.git
```

Go to the project directory

```bash
  cd KEA
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run electron:dev
```
